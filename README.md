# Namu Dark

Korean Night Ink for Vim.

A dark theme inspired by Koreaâ€™s documentary painting tradition.
Clean lines, warm paperless neutrals, and pigment-like accents.

## Install

### Vundle

Add this to your `.vimrc`:

```vim
Plugin 'gg582/namu-dark.vim'
```

Then in Vim:

```vim
:PluginInstall
```

## Use

```vim
colorscheme namu-dark
```

## Design notes

- Background: deep, warm ink tone
- Foreground: soft mineral white
- Keywords: calm jade green for structure
- Strings: blue ink accent
- Types: muted plum
- Numbers and constants: warm ochre
- Functions: gentle teal
